# 🌟 Futuristic Minimal Portfolio

A cutting-edge, experimental portfolio website that breaks the mold with innovative design, interactive elements, and a futuristic aesthetic. Built with modern web technologies and designed to showcase your work in a truly unique way.

![Portfolio Preview](https://via.placeholder.com/800x400/0F0F0F/00FFF7?text=Futuristic+Minimal+Portfolio)

## ✨ Features

### 🎨 **Futuristic Minimal Design**
- **Jet Black Background** (#0F0F0F) with Aqua Cyan accents (#00FFF7)
- **Glassmorphism Effects** with backdrop blur and subtle borders
- **Dynamic Gradient Blobs** that float and respond to scroll
- **Glitch Animations** on hover for that cyberpunk feel

### 🚀 **Interactive Elements**
- **Custom Cursor** with hover effects and particle trail
- **Mouse-Responsive Particles** that follow your movement
- **Horizontal Scroll Projects** with drag/swipe functionality
- **Theme Toggle** with smooth dark/light mode transitions
- **Konami Code Easter Egg** (try ↑↑↓↓←→←→BA!)

### 🎭 **Personality-Driven Content**
- **Terminal Component** in the About section
- **Asymmetrical Layouts** that break traditional grid patterns
- **Personal Voice** throughout the copy ("Who's Behind the Code?", "Things I've Built")
- **Experimental Navigation** with creative section names

### ⚡ **Advanced Animations**
- **Scroll-Triggered Animations** with intersection observer
- **Parallax Effects** on background elements
- **Floating Animations** for dynamic elements
- **Skill Bar Animations** with progress indicators
- **Hover Effects** with depth and glow

### 📱 **Responsive & Accessible**
- **Mobile-First Design** with touch-friendly interactions
- **Smooth Scrolling** navigation
- **Keyboard Navigation** support
- **Screen Reader** friendly
- **Cross-Browser** compatibility

## 🎨 Customization

### Color Schemes
The portfolio comes with three built-in color schemes:

1. **Futuristic Minimal** (Default) - Jet black with cyan accents
2. **Soft Neo-Retro** - Peachy beige with rose and blue
3. **Codepunk** - Deep navy with neon pink and lime

### Easy Personalization
- Replace all `[PLACEHOLDER]` values in `index.html`
- Update colors in the `:root` section of `styles.css`
- Add your images and project screenshots
- Customize the terminal component
- Update contact information and social links

## 🚀 Deployment

## 🎨 Design Philosophy

This portfolio is built on the principle that **form follows function with personality**. Every element serves a purpose while maintaining a unique, memorable aesthetic that reflects the experimental nature of modern web development.

### Key Design Principles:
- **Asymmetry** over rigid grids
- **Depth** through glassmorphism and shadows
- **Movement** through animations and interactions
- **Personality** through voice and micro-interactions
- **Innovation** through experimental features
